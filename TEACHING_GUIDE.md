## 1. 티칭이란?

로봇이 반복할 경로(위치 시퀀스)를 사람이 직접 가르쳐주는 작업입니다.

```
티칭 → 프로그램(포인트 목록) 저장 → 자동 운전으로 반복 재생
```

이 시스템에서 지원하는 티칭 방법은 세 가지입니다:

| 방법 | 특징 | 적합한 상황 |
|------|------|------------|
| **JOG 티칭** | HMI 버튼으로 관절 이동 후 TEACH 클릭 | 정밀 위치, 소수 포인트 |
| **Servo Free 직접 교시** | 서보 끄고 팔을 손으로 밀어 자세 → TEACH | 직관적 경로, 다수 포인트 |
| **수동 입력** | 편집기에 각도 값 직접 입력 | 계산된 위치, 오프라인 작업 |

## 2. 사전 준비

```
1. bridge.py 실행
   python bridge.py --port COM32

2. 브라우저에서 HMI 접속
   http://localhost:5100

3. Connect → "All Ready" 초록 확인

4. [▶ Enable All] 클릭 → LOG에 [AxJ1] OP_ENABLED 확인

5. 수동 운전 탭 선택
```

## 3. 티칭 방법 1 — JOG 티칭 (기본)

HMI의 JOG 버튼으로 각 관절을 이동시키고 [◉ TEACH]로 위치를 기록하는 방법입니다.

### 3.1 워크플로우

```
수동 운전 탭
  ↓
[속도 배율] 선택  →  ×0.01 (미세) / ×0.1 / ×1 / ×10 (빠름)
  ↓
J1~J6 [−] / [+] 버튼으로 원하는 자세로 이동
  ↓
DRO에서 현재 각도 확인 (J1: +45.00° 등)
  ↓
[SYNC / RAPID] 선택 + 이동 시간(DUR) 설정
  ↓
[◉ TEACH] 클릭  →  편집기에 자동 삽입됨
  ↓
반복 (다음 포인트로 JOG → TEACH)
  ↓
[▶ 자동탭] 클릭  →  자동 운전 탭에서 확인 및 실행
```

### 3.2 화면 예시

```
JOG 패널                      TEACH 패널
──────────────────────────    ─────────────────────────────────
속도 배율: ×0.1 [선택]        SYNC  RAPID
J1  [−]  [+]                  이동 시간: 2.0 초
J2  [−]  [+]
J3  [−]  [+]         →→→     [◉  TEACH]  ← 클릭
J4  [−]  [+]                  [↩ 취소] [▶ 자동탭] [🗑 초기화]
J5  [−]  [+]                  포인트 수: 3
J6  [−]  [+]
```

### 3.3 삽입되는 명령 형태

SYNC 모드 선택 시:
```
SYNC J1=+45.00 J2=+30.00 J3=+90.00 J4=+00.00 J5=+45.00 J6=+00.00 DUR=2.0
```

RAPID 모드 선택 시:
```
RAPID J1=+45.00 J2=+30.00 J3=+90.00 J4=+00.00 J5=+45.00 J6=+00.00
```

## 4. 티칭 방법 2 — Servo Free 직접 교시

**✋ Servo Free** 버튼을 누르는 동안 서보가 비활성화되어 팔을 손으로 직접 밀어 자세를 잡을 수 있습니다.

### 4.1 원리

```
버튼 누름 (Hold)
  → 전 관절 서보 Disable (토크 0)
  → 팔을 손으로 자유롭게 이동 가능

버튼 해제 (Release)
  → 서보 자동 Enable 복귀
  → 현재 위치 유지 (드라이브가 위치 잠금)
  → DRO에 새 각도 표시
  → [◉ TEACH] 로 기록
```

### 4.2 워크플로우

```
[▶ Enable All]  →  서보 ON 확인

✋ Servo Free 버튼을 누름 (Hold)
  → 버튼이 빨간색으로 변경 ("🔴 Servo DISABLED")
  → 팔을 손으로 원하는 자세로 이동

버튼에서 손 뗌 (Release)
  → 서보 Enable 자동 복귀 (노란색으로 복귀)
  → DRO에 현재 각도 표시

[◉ TEACH] 클릭  →  포인트 기록

반복
```

### 4.3 주의사항

> ⚠️ **중력이 있는 관절(수직 팔)은 서보 OFF 시 자중으로 떨어질 수 있습니다.**  
> 반드시 팔을 손으로 받쳐서 지지한 상태에서 Free 버튼을 누르세요.

> ⚠️ **드라이브가 토크 제어(Torque Mode)를 지원해야 합니다.**  
> LS L7NH는 CSP 모드이므로 Free 시 드라이브가 SWITCHED_ON 또는 READY 상태로 전환됩니다.

## 5. 티칭 방법 3 — 수동 입력 (오프라인)

관절 각도를 미리 계산하여 자동 운전 탭 편집기에 직접 입력합니다.

### 5.1 워크플로우

```
자동 운전 탭 → 편집기에 직접 입력

예시:
  ; 홈 위치
  SYNC J1=+0.00 J2=+0.00 J3=+0.00 J4=+0.00 J5=+0.00 J6=+0.00 DUR=3.0

  ; 작업 위치 1
  SYNC J1=+45.00 J2=+30.00 J3=+90.00 DUR=2.0

  ; 작업 위치 2
  RAPID J4=+90.00 J5=+45.00

  ; 복귀
  SYNC J1=+0.00 J2=+0.00 J3=+0.00 J4=+0.00 J5=+0.00 J6=+0.00 DUR=3.0
```

## 6. TEACH 패널 상세

수동 운전 탭 → 중앙 패널 → JOG 버튼 아래에 위치합니다.

### 6.1 구성 요소

| 컨트롤 | 설명 |
|--------|------|
| **SYNC** | 전 관절이 동시에 출발·도착 (5차 다항식) |
| **RAPID** | 각 관절이 독립적으로 최대 속도로 이동 |
| **이동 시간 (초)** | SYNC 명령의 DUR 값 (기본 2.0초) |
| **◉ TEACH** | 현재 J1~J6 각도를 자동 운전 편집기에 삽입 |
| **↩ 취소** | 마지막으로 삽입된 포인트 1개 삭제 |
| **▶ 자동탭** | 자동 운전 탭으로 전환하여 내용 확인 |
| **🗑 초기화** | 편집기 전체 내용 삭제 (확인 창) |
| **N 포인트** | 현재 세션에서 기록한 포인트 수 |

### 6.2 SYNC vs RAPID 선택 기준

| 상황 | 권장 |
|------|------|
| 부드럽고 자연스러운 동작 | **SYNC** |
| 정밀한 속도 제어 필요 | **SYNC** (DUR 조정) |
| 빠른 위치 이동, 경로 무관 | **RAPID** |
| 혼합 사용 | SYNC와 RAPID를 섞어도 됨 |

## 7. Servo Free 버튼 상세

수동 운전 탭 → 우측 패널 → Quick Actions에 위치합니다.

### 7.1 버튼 상태

| 상태 | 색상 | 서보 | 동작 |
|------|------|------|------|
| **대기** | 노란색 테두리 | Enable | — |
| **누름 중** | 빨간색 글로우 | **Disable** | 팔 자유 이동 가능 |
| **해제** | 노란색 복귀 | Enable 자동 복귀 | 위치 유지 |

### 7.2 버튼 해제 시 동작 규칙

```
버튼 누르기 전 상태  →  해제 후 상태
─────────────────────────────────────
Enable                →  Enable 자동 복귀  ✅
Disable               →  Disable 유지      (원래 상태 유지)
```

### 7.3 입력 방법

- **마우스:** 버튼 클릭 유지 → 해제
- **터치:** 터치 유지 → 뗌
- **안전:** 마우스가 버튼 밖으로 이탈해도 자동 해제됨 (`mouseleave`)

## 8. 자동 운전으로 재생

### 8.1 재생 순서

```
자동 운전 탭
  ↓
편집기에서 프로그램 확인 및 수정
  ↓
[▶ 실행] 클릭
  ↓
실행 오버레이:
  - 현재 실행 줄: 노란색 강조
  - 완료된 줄: 회색 처리
  - 오류 줄: 빨간색
  ↓
자동으로 DONE → ACK → 다음 줄 순차 실행
  ↓
완료 → "✏ 편집" 버튼으로 편집 모드 전환
```

### 8.2 실행 중 제어

| 버튼 | 동작 |
|------|------|
| **⏹ 정지** | 즉시 중단 (E-STOP 전송) |
| **↩ 처음으로** | 커서를 첫 줄로 초기화 |
| **✏ 편집** | 완료 후 편집 모드 전환 |

### 8.3 파일 저장

```
[💾 저장]      → 현재 편집기 내용을 .nc 파일로 저장
[📂 열기]      → 저장된 파일 불러오기
[💾 다른이름]  → 새 파일명으로 저장
```

## 9. 티칭 팁 & 주의사항

### ✅ 권장 사항

**포인트 순서 설계**
```
항상 홈 위치 → 작업 위치들 → 홈 복귀 순으로 구성
예:
  SYNC J1=0 ... DUR=3.0         ; 시작: 홈
  SYNC J1=45 J2=30 ... DUR=2.0  ; 작업 위치 1
  SYNC J1=90 J2=60 ... DUR=2.0  ; 작업 위치 2
  SYNC J1=0 ... DUR=3.0         ; 종료: 홈 복귀
```

**이동 시간 설정**
```
빠른 이동: DUR=1.0~1.5 초
일반 이동: DUR=2.0~3.0 초  (기본값)
느린 이동: DUR=4.0~5.0 초  (정밀 작업)
```

**JOG 속도 배율 선택**
```
×0.01 — 0.01 × profile_velocity, 극초정밀 미세 조정
×0.1  — 정밀 조정
×1    — 일반 JOG (기본)
×10   — 큰 이동 (충돌 주의)
```

### ⚠️ 주의사항

1. **티칭 전 작업 영역 확보**  
   로봇 이동 반경 내 사람과 장애물이 없는지 확인하세요.

2. **첫 재생은 저속으로**  
   티칭 직후 첫 자동 운전은 RAPID 대신 SYNC + 긴 DUR로 천천히 확인하세요.

3. **소프트 리밋 설정 권장**  
   ⚙ 파라미터에서 각 관절의 + / − 리밋을 설정하면 작업 범위를 벗어나는 이동을 방지합니다.

4. **Flash 저장**  
   파라미터(unit_scale, limits 등) 변경 후 반드시 [💾 Save Flash]로 저장하세요.  
   **티칭 프로그램 자체는 HMI 로컬 파일로 저장**하세요 ([💾 저장] 버튼).

5. **Servo Free 사용 시**  
   중력 영향을 받는 관절은 손으로 받친 상태에서만 Free 버튼을 누르세요.
